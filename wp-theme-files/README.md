# WordPress Theme Files — Pricing Landing Page

These are the active WordPress theme files for the Epipheo pricing page at `epipheo.com/explainer-video-price/`.

**Theme:** minimo-child

| File | Location on Server | Purpose |
|------|--------------------|---------|
| `page-landing.php` | `wp-content/themes/minimo-child/page-landing.php` | Custom page template (does not call `the_content()`) |
| `landing.js` | `wp-content/themes/minimo-child/assets/js/landing.js` | Page JS, loaded as `type="module"` via `script_loader_tag` filter |
| `landing.css` | `wp-content/themes/minimo-child/assets/css/landing.css` | Page CSS |
| `functions.php` | `wp-content/themes/minimo-child/functions.php` | Theme functions, registers/enqueues assets |
